# TC-004: Неуспешная авторизация с некорректным номером телефона

## ID тест-кейса
TC-004

## Название
Проверка ошибки при вводе некорректного номера телефона при авторизации

## Приоритет
Средний

## Предусловия

- Пользователь находится на странице авторизации

## Шаги выполнения

| Шаг | Действие | Ожидаемый результат | Фактический результат | Статус |
|---|---|---|---|---|
| 1 | Открыть сайт https://sbermarket.ru | Сайт успешно открылся. Отображается кнопка «Войти» | Соответствует ожидаемому результату | Пройден |
| 2 | Нажать кнопку «Войти» | Открывается форма авторизации | Соответствует ожидаемому результату | Пройден |
| 3 | Ввести некорректный номер телефона | Номер введён в поле авторизации | Соответствует ожидаемому результату | Пройден |
| 4 | Нажать кнопку «Получить код в SMS» | Система отображает сообщение об ошибке о некорректном формате номера | Отображается сообщение: «Номер должен начинаться с +7 (9..» | Пройден |

## Результат тестирования

Пройден

# TC-001: Успешная авторизация с помощью SMS

## ID тест-кейса
TC-001

## Название
Успешная авторизация пользователя через SMS-код

## Приоритет
Высокий

## Предусловия

- Пользователь имеет валидный номер телефона

## Шаги выполнения

| Шаг | Действие | Ожидаемый результат | Фактический результат | Статус |
|---|---|---|---|---|
| 1 | Открыть сайт https://sbermarket.ru | Сайт успешно открылся. В правом верхнем углу отображается кнопка «Войти» | Соответствует ожидаемому результату | Пройден |
| 2 | Нажать кнопку «Войти» | Открывается форма авторизации | Соответствует ожидаемому результату | Пройден |
| 3 | Ввести валидный номер телефона | Номер успешно принят системой | Соответствует ожидаемому результату | Пройден |
| 4 | Нажать кнопку «Получить код в SMS» | Отображается поле для ввода шестизначного кода | Соответствует ожидаемому результату | Пройден |
| 5 | Ввести код из SMS | Пользователь успешно авторизован | Соответствует ожидаемому результату | Пройден |

## Результат тестирования

Пройден

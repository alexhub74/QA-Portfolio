# TC-008: Удаление товара из корзины

## ID тест-кейса
TC-008

## Название
Проверка удаления товара из корзины

## Приоритет
Средний

## Предусловия

- Открыт сайт https://sbermarket.ru
- Пользователь авторизован
- В корзине находится добавленный товар

## Шаги выполнения

| Шаг | Действие | Ожидаемый результат | Фактический результат | Статус |
|---|---|---|---|---|
| 1 | Открыть корзину | Отображается список добавленных товаров | Соответствует ожидаемому результату | Пройден |
| 2 | Нажать кнопку удаления товара | Товар удаляется из корзины | Соответствует ожидаемому результату | Пройден |
| 3 | Проверить содержимое корзины | Удалённый товар отсутствует | Соответствует ожидаемому результату | Пройден |

## Результат тестирования

Пройден

# TC-005: Авторизация через СберБизнес ID

## ID тест-кейса
TC-005

## Название
Проверка авторизации пользователя через СберБизнес ID

## Приоритет
Высокий

## Предусловия

- Пользователь находится на странице авторизации
- У пользователя есть данные для входа в СберБизнес ID

## Шаги выполнения

| Шаг | Действие | Ожидаемый результат | Фактический результат | Статус |
|---|---|---|---|---|
| 1 | Открыть сайт https://sbermarket.ru | Сайт успешно открылся. Отображается кнопка «Войти» | Соответствует ожидаемому результату | Пройден |
| 2 | Нажать кнопку «Войти» | Открывается форма авторизации | Соответствует ожидаемому результату | Пройден |
| 3 | Установить чекбокс «Хочу заказывать для бизнеса» | Появляется кнопка «Войти по СберБизнес ID» | Соответствует ожидаемому результату | Пройден |
| 4 | Нажать кнопку «Войти по СберБизнес ID» | Открывается страница авторизации СберБизнес ID | Соответствует ожидаемому результату | Пройден |
| 5 | Ввести логин | Логин успешно введён | Соответствует ожидаемому результату | Пройден |
| 6 | Ввести пароль | Пароль успешно введён | Соответствует ожидаемому результату | Пройден |
| 7 | Нажать кнопку «Далее» | Пользователь успешно авторизуется | Отображается ошибка «Логин или пароль неверны» | Не пройден |

## Результат тестирования

Не пройден

## Найденный дефект

BUG-001: Ошибка авторизации через СберБизнес ID

# TC-002: Успешная авторизация через Сбер ID по QR-коду

## ID тест-кейса
TC-002

## Название
Авторизация пользователя через Сбер ID с использованием QR-кода

## Приоритет
Высокий

## Предусловия

- У пользователя установлено приложение СберБанк Онлайн

## Шаги выполнения

| Шаг | Действие | Ожидаемый результат | Фактический результат | Статус |
|---|---|---|---|---|
| 1 | Открыть сайт https://sbermarket.ru | Сайт успешно открылся. Отображается кнопка «Войти» | Соответствует ожидаемому результату | Пройден |
| 2 | Нажать кнопку «Войти» | Открывается форма авторизации | Соответствует ожидаемому результату | Пройден |
| 3 | Нажать кнопку «Войти по Сбер ID» | Открывается форма авторизации через Сбер ID | Соответствует ожидаемому результату | Пройден |
| 4 | Выбрать вход по QR-коду | Отображается QR-код для сканирования | Соответствует ожидаемому результату | Пройден |
| 5 | Отсканировать QR-код через приложение СберБанк Онлайн | На смартфоне появляется страница подтверждения входа | Соответствует ожидаемому результату | Пройден |
| 6 | Подтвердить вход на смартфоне | На компьютере выполняется авторизация и открывается главная страница | Соответствует ожидаемому результату | Пройден |

## Результат тестирования

Пройден

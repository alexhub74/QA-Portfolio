# TC-003: Успешная авторизация через Сбер ID по номеру телефона

## ID тест-кейса
TC-003

## Название
Авторизация пользователя через Сбер ID с использованием номера телефона

## Приоритет
Высокий

## Предусловия

- Пользователь имеет валидный номер телефона

## Шаги выполнения

| Шаг | Действие | Ожидаемый результат | Фактический результат | Статус |
|---|---|---|---|---|
| 1 | Открыть сайт https://sbermarket.ru | Сайт успешно открылся. В правом верхнем углу отображается кнопка «Войти» | Соответствует ожидаемому результату | Пройден |
| 2 | Нажать кнопку «Войти» | Открывается форма авторизации | Соответствует ожидаемому результату | Пройден |
| 3 | Нажать кнопку «Войти по Сбер ID» | Открывается форма авторизации через Сбер ID | Соответствует ожидаемому результату | Пройден |
| 4 | Ввести валидный номер телефона | Номер успешно принят системой | Соответствует ожидаемому результату | Пройден |
| 5 | Нажать кнопку «Войти или создать Сбер ID» | Открывается страница для ввода кода подтверждения | Соответствует ожидаемому результату | Пройден |
| 6 | Ввести код из SMS | Пользователь успешно авторизован, открывается главная страница | Соответствует ожидаемому результату | Пройден |

## Результат тестирования

Пройден

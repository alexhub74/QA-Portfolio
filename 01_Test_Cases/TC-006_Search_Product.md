# TC-006: Поиск товара по названию

## ID тест-кейса
TC-006

## Название
Проверка поиска товара по названию

## Приоритет
Средний

## Предусловия

- Открыт сайт https://sbermarket.ru
- Выполнена авторизация
- Пользователь находится на главной странице

## Шаги выполнения

| Шаг | Действие | Ожидаемый результат | Фактический результат | Статус |
|---|---|---|---|---|
| 1 | Ввести название товара в поисковую строку | Текст отображается в поле поиска | Соответствует ожидаемому результату | Пройден |
| 2 | Нажать кнопку поиска | Открывается страница результатов поиска | Соответствует ожидаемому результату | Пройден |
| 3 | Проверить список найденных товаров | Отображаются товары, соответствующие поисковому запросу | Соответствует ожидаемому результату | Пройден |

## Результат тестирования

Пройден

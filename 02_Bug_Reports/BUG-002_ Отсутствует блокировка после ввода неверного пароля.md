# BUG-002: Отсутствует ограничение попыток авторизации

## Название
Пользователь может многократно вводить неверный пароль без ограничения количества попыток входа.

## Окружение
- ОС: Windows
- Браузер: Google Chrome
- Сайт: https://sbermarket.ru

## Предусловия
Пользователь зарегистрирован в системе и находится на странице авторизации.

## Шаги воспроизведения
1. Открыть страницу авторизации.
2. Ввести корректный логин.
3. Ввести неверный пароль.
4. Нажать кнопку «Войти».
5. Повторить попытку входа несколько раз.

## Ожидаемый результат
После нескольких неудачных попыток система должна ограничить количество попыток входа или показать уведомление о временной блокировке.

## Фактический результат
Пользователь может продолжать вводить неверный пароль без ограничений.

## Severity
Medium

## Priority
Medium

## Статус

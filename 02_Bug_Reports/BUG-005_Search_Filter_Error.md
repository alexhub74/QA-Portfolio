# BUG-005: Некорректная работа фильтра поиска

## Название

Фильтр товаров отображает результаты, которые не соответствуют выбранным параметрам.

## Окружение

- ОС: Windows
- Браузер: Google Chrome
- Сайт: https://sbermarket.ru

## Предусловия

Пользователь находится на странице каталога товаров.

## Шаги воспроизведения

1. Открыть каталог товаров.
2. Выбрать любой фильтр (категория, цена или другие параметры).
3. Нажать кнопку применения фильтра.
4. Проверить отображаемые результаты.

## Ожидаемый результат

В списке отображаются только товары, соответствующие выбранным параметрам фильтра.

## Фактический результат

В списке отображаются товары, которые не соответствуют выбранным параметрам.

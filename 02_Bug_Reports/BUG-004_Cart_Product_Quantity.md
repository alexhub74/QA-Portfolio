# BUG-004: Количество товара в корзине не сохраняется

## Название

После изменения количества товара в корзине значение сбрасывается.

## Окружение

- ОС: Windows
- Браузер: Google Chrome
- Сайт: https://sbermarket.ru

## Предусловия

В корзину пользователя добавлен товар.

## Шаги воспроизведения

1. Открыть корзину.
2. Изменить количество товара с 1 на 2.
3. Обновить страницу.
4. Проверить количество товара.

## Ожидаемый результат

После обновления страницы отображается изменённое количество товара.

## Фактический результат

После обновления страницы отображается первоначальное количество товара.

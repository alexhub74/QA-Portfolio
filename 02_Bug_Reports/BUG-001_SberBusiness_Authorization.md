# BUG-001: Ошибка авторизации через СберБизнес ID

## Название
Невозможность авторизации через СберБизнес ID с указанными данными

## Окружение
* ОС: Windows
* Браузер: Google Chrome
* Сайт: https://sbermarket.ru

## Предусловия
Пользователь находится на странице авторизации СберМаркет.

## Шаги воспроизведения

1. Открыть сайт https://sbermarket.ru
2. Нажать кнопку «Войти»
3. Установить чекбокс «Хочу заказывать для бизнеса»
4. Нажать кнопку «Войти по СберБизнес ID»
5. Ввести логин: `Bisnessmen1`
6. Ввести пароль: `123321`
7. Нажать кнопку «Далее»

## Ожидаемый результат
Пользователь успешно авторизуется и переходит на главную страницу сайта.

## Фактический результат
Отображается сообщение: > «Логин или пароль неверны»
Авторизация пользователя не выполняется.

## Severity
Major
## Priority
High
## Статус
Open

## Комментарий
Ошибка обнаружена при выполнении функционального тестирования сценария авторизации через СберБизнес ID.
